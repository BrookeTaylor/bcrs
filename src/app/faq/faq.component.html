Frequently Asked Questions

Here you will find step-by-step instructions to complete necessary work functions here at Bob's Computer Repair Shop.

View the user list

  1. To view the list of user's, click on the profile at the top of any page.



  2. Then click on the users option to take you to the user list page.



Create a new user.

  1. Click on the new button "New User" button above the user list.



  2. Fill out the new user's info then click submit.



Edit a current user.

  1. Click on the edit icon for the user you wish to edit.



  2. Edit the user's info field(Note: Name and Role are the only editable parts for the user).



Delete a user.

  1. Click on the trash icon for the user you want to delete.



  2. On the popup, click OK to delete the user(Note: Once the user is deleted, they can not be brought back).
